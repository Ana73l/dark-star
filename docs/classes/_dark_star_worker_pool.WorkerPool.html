createTask

  • createTask<TData, TResult>(executable: ((data: TData) => TResult)): TaskRunner<TData, TResult>

  • Create a task factory. Tasks scheduled from a single factory are unique independent instance and thus can be scheduled multiple times.

    Returns

    • Task factory that will be used to schedule tasks using a common callback
    // recursive inefficient implementation
    function fibonacci(input: number): number {
        return input < 2 ? input : fibonacci(input - 2) + fibonacci(input - 1);
    }

    const pool = new WorkerPool({ threads: 3 });
    const runFibonacci = workerPool.createTask(fibonacci);
    `;
